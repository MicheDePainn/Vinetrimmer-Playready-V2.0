1. Install Python 3.8 - 3.11 (Python 3.12+ might have issues with some older dependencies, but 3.8-3.11 are recommended). Be sure to add Python to PATH while installing it.

2. Install Microsoft Visual C++ Redistributable:
https://aka.ms/vs/17/release/vc_redist.x64.exe

3. Run install.bat to install poetry and dependencies.

4. For services requiring credentials (e.g. DisneyPlus, iTunes, ParamountPlus), open vinetrimmer\vinetrimmer.yml (with Notepad++) and add your email and password under the "credentials:" section. You do not need to add credentials for services that rely solely on cookies (like Amazon, Peacock, Hulu, CanalPlus, Max, NowTV, TimVision), just the cookies file. The cookies file name will always be default.txt.

5. Install this Firefox add-on:
https://addons.mozilla.org/en-US/firefox/addon/cookies-txt-one-click/
or for Chrome install this extension:
https://chrome.google.com/webstore/detail/open-cookiestxt/gdocmgbfkjnnpapoeobnolbbkoibbcif

Go to the streaming website, log in, download the cookies, rename the file to default.txt and put it in the corresponding folder:
- For Amazon: vinetrimmer\cookies\Amazon\default.txt
- For Peacock: vinetrimmer\cookies\Peacock\default.txt
- For Hulu: vinetrimmer\cookies\Hulu\default.txt
- For AppleTV+: vinetrimmer\cookies\AppleTVPlus\default.txt
- For Canal+: vinetrimmer\cookies\Canalplus\default.txt
- For Max: vinetrimmer\cookies\Max\default.txt
- For NowTV: vinetrimmer\cookies\NowTV\default.txt
- For TimVision: vinetrimmer\cookies\Timvision\default.txt

6. You can download videos by using the batch scripts in the main folder (e.g., download.Amazon.bat, download.DisneyPlus.bat, etc.), or use the command line directly.

Examples of direct command line usage:

For downloading the video of an episode from Prime Video use:
poetry run vt dl -al en -sl en -w S01E1 Amazon -vq SD https://www.primevideo.com/region/eu/detail/0KRGHGZCHKS920ZQGY5LBRF7MA/
To download the video for a specific season:
poetry run vt dl -al en -sl en -w S01 Amazon -vq SD https://www.primevideo.com/region/eu/detail/0KRGHGZCHKS920ZQGY5LBRF7MA/

7. For downloading the video of an episode from Peacock use:
poetry run vt dl -al en -w S01E1 Peacock https://www.peacocktv.com/watch/asset/tv/yellowstone/7034893369976297112
For a movie:
poetry run vt dl Peacock -m https://www.peacocktv.com/watch/asset/movies/john-wick/a698e22c-3e32-35cc-9a0c-981b90402fc2

8. For downloading a movie from DisneyPlus use:
poetry run vt dl -q 1080 DisneyPlus -m https://www.disneyplus.com/en-gb/browse/entity-3d00a2de-821a-4844-ad90-2f9e7b685b95

9. For downloading the video of a movie from Hulu use:
poetry run vt dl Hulu -m https://www.hulu.com/movie/ce42e744-f65d-4d46-9324-80aec621fc87

10. For downloading a movie from ParamountPlus use:
poetry run vt dl ParamountPlus -m https://www.paramountplus.com/movies/video/7juDj4xQPZG1xtvD0pDPRQWgv1Vj6xox/
For an episode:
poetry run vt dl -w S01E1 ParamountPlus https://www.paramountplus.com/shows/1883/

11. For AppleTVPlus you need to use the correct value of 'sf' in the file vinetrimmer\config\services\appletvplus.yml for the country of your account. You can get it from this file:
https://gist.githubusercontent.com/BrychanOdlum/2208578ba151d1d7c4edeeda15b4e9b1/raw/8f01e4a4cb02cf97a48aba4665286b0e8de14b8e/storefrontmappings.json
(Also for iTunes you can use the -sf parameter).

12. For Skyshowtime add the country code of the account in vinetrimmer\config\services\skyshowtime.yml in territory and activeterritory.

13. For France.tv you do not need cookies or credentials for free content. The script uses PlayReady DRM to download and decrypt the videos.
Example for an episode:
poetry run vt dl -w S16E01 FranceTV https://www.france.tv/france-3/lego-ninjago/

For more options use:
poetry run vt dl -h

14. To use your own PlayReady CDM, create a new folder in vinetrimmer\devices (e.g., vinetrimmer\devices\my_cdm_sl3000).
Copy your PlayReady group key (zgpriv.dat) and group certificate (bgroupcert.dat) into this folder.
Create a new text file named PR.json in that folder and add something like:

{
  "device": "my_cdm.prd",
  "SecurityLevel": 3000
}

Change the SecurityLevel to match your CDM (e.g., 2000 or 3000).
Open the file vinetrimmer\vinetrimmer.yml with Notepad++ and set your new CDM as default:

cdm:
  default: 'my_cdm_sl3000'
